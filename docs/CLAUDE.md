# ラーメンレビューサイト

## 目的

食べログの保存リストを管理する個人サイト

## 技術構成

- HTML
- CSS
- JavaScript
- GitHub Pages

## 必須機能

- ラーメン店一覧表示
- 店舗検索
- タグ検索
- お気に入り表示

## デザイン

- ポップ
- Bentoレイアウト
- 緑を基調
- ラーメンモチーフ

## データ

ramen-data.json を使用

## ルール

- レスポンシブ対応
- JavaScriptはES6
- コメントは日本語
- 作業開始時はtodo.mdに作業内容を記載
- 作業終了後に完了内容をtodo.mdとdecisions.mdを更新する
